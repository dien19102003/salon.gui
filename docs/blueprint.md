# **App Name**: Shear Bliss

## Core Features:

- Homepage: Elegant landing page with service categories, salon intro, reviews, and a booking button.
- Service Catalog: Detailed service cards with names, durations, prices, and photos.
- Service Details and Booking: In-depth service descriptions, stylist selection, and time slot availability, with a clear 'Book Now' call-to-action.
- Step-by-step Booking Flow: Streamlined booking process: service → stylist → time → personal info → confirmation.
- Stylist Profiles: Pages for each stylist with photos, skills, ratings, and schedules.
- Customer Profile: Personalized profile with booking history, profile editing, and appointment overviews.
- Intelligent Schedule Management: Use an LLM to automatically suggest optimal booking times based on stylist availability and resource allocation.
- Booking Management Dashboard: Backend interface with booking CRUD operations, status updates, filters, and calendar view. Firebase

## Style Guidelines:

- Primary color: Soft dark green (#4A7A53) to align with a natural and premium feel.
- Background color: Light, desaturated green (#E8F0E7) for a clean and minimal look.
- Accent color: Muted gold (#B3A369) for highlighting interactive elements.
- Body and headline font: 'PT Sans', a sans-serif for high readability and a clean, professional look.
- Simple, elegant line icons for service categories and UI elements.
- Use of soft shadows and rounded corners to create a modern and approachable interface.
- Subtle transitions and animations to enhance the user experience.
